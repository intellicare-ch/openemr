# Document Management

**Capability ID:** 13
**Last Updated:** 2025-11-15

## Overview
Comprehensive document storage and management with multi-category organization, scanning, templates, patient linking, signing, DICOM viewer, and secure sharing.

## Key Features
- Multi-category document organization
- Document upload and scanning
- Template management
- Patient document linking
- Electronic document signing
- DICOM viewer for medical imaging
- PDF generation
- Document access control
- Version tracking
- Secure document sharing

**Files:** `/controllers/C_Document.class.php`, `/interface/patient_file/summary/documents.php`
**Tables:** `documents`, `categories`, `categories_to_documents`
**Service:** `/src/Services/DocumentService.php`

## Configuration
- Document storage location
- Allowed file types
- Maximum file sizes
- Category permissions
- Encryption settings
