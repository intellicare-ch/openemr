# API and Integration

**Capability ID:** 17
**Last Updated:** 2025-11-15

## Overview
Comprehensive API capabilities including RESTful API, FHIR R4 API, OAuth2 authentication, SMART on FHIR, bulk data export, HL7 interfaces, EDI transactions, and webhook support.

## Key Features
- RESTful API (JSON)
- FHIR R4 API (40+ resources)
- OAuth2 authentication and authorization
- SMART on FHIR app launch
- Bulk FHIR export ($export)
- HL7 v2.x interfaces (ORU, ORM, ADT)
- X12 EDI transactions (270, 271, 835, 837)
- Direct messaging integration
- Custom API endpoints
- Webhook support
- API rate limiting
- API documentation
- Postman collections

**Files:** `/apis/`, `/_rest_routes.inc.php`, `/src/RestControllers/`, `/src/RestControllers/FHIR/`
**Standards:** REST, FHIR R4, HL7 v2.x, X12 EDI, OAuth 2.0, OIDC
