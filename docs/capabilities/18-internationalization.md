# Multi-language Support (Internationalization)

**Capability ID:** 18
**Last Updated:** 2025-11-15

## Overview
International deployment support with 70+ languages, right-to-left language support, localized date/time formats, currency handling, and custom translation management.

## Key Features
- 70+ supported languages
- Right-to-left (RTL) language support (Arabic, Hebrew)
- User-selectable language preference
- Translation string management
- Language constant editing
- Date and time format localization
- Number format localization
- Currency handling
- Custom language additions
- Backend (PHP) and frontend (JavaScript) i18n

**Files:** `/interface/language/`, i18next integration
**Tables:** `lang_definitions`, `lang_constants`
**Functions:** `xl()` for PHP translation, i18next for JavaScript

## Configuration
- Default language
- Available languages
- Date/time formats
- Number formats

**End of Internationalization Capability**
