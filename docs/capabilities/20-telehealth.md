# Telehealth

**Capability ID:** 20
**Last Updated:** 2025-11-15

## Overview
Integrated telehealth capabilities with video conferencing, virtual visit scheduling, patient and provider portals, screen sharing, and calendar integration for remote care delivery.

## Key Features
- Video conferencing
- Virtual visit scheduling
- Provider telehealth portal
- Patient telehealth portal
- Screen sharing
- Waiting room
- Session recording (optional)
- Calendar integration
- Telehealth encounter documentation
- Billing for telehealth visits
- Invitation management

**Module:** `/interface/modules/custom_modules/oe-module-comlink-telehealth/`
**Integration:** Third-party video conferencing platforms

## Configuration
- Telehealth provider selection
- Video quality settings
- Recording options
- Scheduling integration

**End of Telehealth Capability**
