# Customization and Extensibility

**Capability ID:** 19
**Last Updated:** 2025-11-15

## Overview
Extensible architecture supporting custom modules, event-driven hooks, layout-based form builder, custom reports, theme customization, and third-party module installation.

## Key Features
- Module system (Zend and custom modules)
- Event-driven architecture (Symfony EventDispatcher)
- Layout-Based Form (LBF) builder
- Custom report development
- Theme and template customization
- Plugin architecture
- Composer package management
- Custom menu items
- Override system templates
- Custom API endpoints
- Database hooks
- Third-party integrations

**Files:** `/interface/modules/`, `/src/Events/`, `/interface/super/edit_layout.php`
**Module Types:** Zend modules, custom modules, menu modules

## Configuration
- Module registration and activation
- Event subscriber registration
- Custom theme selection
- Module permissions
